FlightR

FlightR is a system that is able to correlate flight dates with the time of flights and ticket prices. This project was completed at the HackReduce Travel Hackathon, and was awarded the most technically interesting project award for our submission. The data analysis was done using the statistical package R, and the visualization was done using d3.js.

  • Code: github/flightr
  • Winning Submission: Winner and Award for Most Technically Interesting Project
